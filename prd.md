# Portfolio V2 - Product Requirements Document

## Overview
A high-end, developer-focused portfolio for Muhammad Asad Khan, a CS student at UBIT. The site showcases technical skills, projects, and live activity through various widgets and integrations.

## Target Audience
- Recruiters and Hiring Managers
- Potential collaborators
- Tech community

## Key Features
1. **Hero Section**: 3D Micro-Orb, typewriter effect, and quick links.
2. **Activity Hub**: Rotating widgets for current reading, movie watching (Letterboxd), TV tracking (Trakt), and football scores (FC Barcelona).
3. **Tech Stack**: Categorized domain-based skill display.
4. **My Big Threes**: Rotating social widget for top movies, shows, players, and artists.
5. **Visitor XP**: Gamified visit counter.
6. **GitHub Heatmap**: Contribution tracker.
7. **Coding Clock**: Displays local time and current activity zone in PKT.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Build Tool**: Vite
- **Deployment**: Vercel
- **APIs**: GitHub, Trakt, Letterboxd (RSS), Last.fm, ESPN (via ESPNs API), TheSportsDB, Wikipedia.

## Design Principles
- **Aesthetic**: liquid-glass, dark mode, high-contrast accents (cyan, purple).
- **Interactions**: Magnetic buttons, scroll-reveal animations, 3D CSS transforms.
- **Performance**: Lightweight, minimum dependencies.
