Asaaniyat (عسانیت)

Mobile-First Agentic AI Orchestrator

The Idea

A mobile-first orchestrator built for the **Google Antigravity Hackathon (Challenge 2)** designed to organize Pakistan's informal economy (handymen, electricians, plumbers). It bridges the gap between fragmented phone calls and digital efficiency, allowing tradespeople to be found and booked seamlessly through advanced AI agent orchestration.

System Capabilities

  • Roman Urdu Voice Parsing: Transcribes and decodes natural spoken dialects into structured intents.
  • Geospatial Neighborhood Resolver: Converts informal location indicators into exact coordinates.
  • Light Mint Glassmorphism: Stunning, native-feeling mobile components styled with frosted-glass sheets.
  • Real-time Telemetry: An operations dashboard tracing each agent pipeline execution dynamically.

The 8-Agent AI Pipeline Flow

Asaaniyat replaces simple, brittle wrappers with an advanced sequential AI pipeline. A centralized state manager tracks request transitions across eight specialized, decoupled micro-agents: 

graph TD
    Client[Client Voice/Text 🗣️] -->|API Request| Parser[1. IntentParser 🧠]
    Parser -->|Service & Intent| Resolver[2. LocationResolver 🗺️]
    Resolver -->|Coordinates| Discoverer[3. ProviderDiscoverer 🔍]
    Discoverer -->|Provider Pool| Ranker[4. ProviderRanker 📊]
    Ranker -->|Ranked Pros| Decision[5. DecisionMaker ⚖️]
    Decision -->|Elected Provider| Pricing[6. DynamicPricing 💰]
    Pricing -->|Final Quotation| Executor[7. BookingExecutor ⚙️]
    Executor -->|Unique Booking ID| FollowUp[8. FollowUpManager 📡]
    FollowUp -->|Surveys & FCM Triggers| Success[Success Confirmation 🎉]

Core Application Technology Stack

React Native (Expo 54) Node.js / Express MongoDB Database Llama 3.3 & Gemini AI React / Vite Admin Google Cloud Run

Technical Achievements

  • Multilingual Natural Language Extraction: Decodes and processes conversational speech transcripts in standard Urdu, Roman Urdu, and English into clean programmatic models without depending on static regexes.
  • Decoupled State Orchestrator: Shared transaction state context tracking, enabling sequential execution logs, performance telemetries, and automated transaction recovery.
  • Scalable Cloud Architecture: Backend microservices packaged via Docker and deployed dynamically to Google Cloud Run, backed by a production-grade MongoDB cloud database, with an administration control interface hosted on Vercel.
