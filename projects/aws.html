AWS EC2 Deployment

Static Website & Cloud Infrastructure

Infrastructure Architecture & Mechanics

A foundational cloud hosting project aimed at hands-on deployment mechanics over native Linux server environments. Involved root-level manual provisioning bridging VPC networking, instance scaling limitations, and direct Nginx routing methodologies.

Core Deployments

  • Manual EC2 Provisioning: Created and maintained an active `t3.micro` Ubuntu instance natively within Linux.
  • Protocol Restrictions: Designed precise Inbound/Outbound ICMP and TCP mapping rules preserving strict closed-server security via AWS SG.
  • SSH Terminal Binding: Bound explicit root authorization certificates utilizing encrypted `.pem` private keys.
  • Nginx Routing: Bootstrapped and configured Nginx configurations (`sites-enabled`), routing to the standard `/var/www/html` directory securely.
